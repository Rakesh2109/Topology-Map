          🧹 Data Taming Pipeline
        

        

          Raw IoMT simulator output needs preprocessing before ML training.
          The Data Tamer runs a 6-step pipeline: clean → cap → select → normalise → balance → export.
        


        
        

          

            
MinMax Scaler

            
x' = (x − min) / (max − min)

            
Fast but sensitive to outliers. Good when few attack spikes present.

          

          

            
Standard Scaler

            
x' = (x − μ) / σ

            
Good for normal distributions. Attack spikes shift the mean.

          

          

            
Robust Scaler ★ Recommended

            
x' = (x − median) / IQR

            
Uses median/IQR — attack outliers do NOT distort the scale.

🐍 Python Usage

        
from data_tamer import DataTamer, tame_csv_file


# One-shot: read raw CSV, tame it, write output

tame_csv_file(

  input_path="output/windows_train.csv",

  output_path="output/tamed_train.csv",

  strategy="robust",       # minmax | standard | robust

  balance="undersample",   # none | undersample | oversample | hybrid

  config_path="output/tamer_config.json",

  report_path="output/taming_report.txt",

)


# Or step by step:

tamer = DataTamer(strategy="robust", balance="hybrid")

tamed = tamer.fit_transform(windows_list)   # list of dicts

tamer.to_csv(tamed, "output/tamed.csv")

tamer.save_config("output/tamer_config.json")

print(tamer.feature_summary())


# Apply saved config to new inference data (no re-fitting):

tamer2 = DataTamer()

tamer2.load_config("output/tamer_config.json")

live_tamed = tamer2.transform(live_windows)
